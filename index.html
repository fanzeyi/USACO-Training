USACO Training 中文翻译

本项目旨在提供一个排版精美的 USACO Training 题目翻译，翻译内容均来自 NOCOW 中的相关词条。

本项目应用了来自 sofish 贡献的 typo.css 项目，感谢他为中文网页排版做出的杰出贡献。

在浏览此项目的时候我们建议您使用最新版的 Chrome, Safari, Firefox 浏览器，以获得最佳的阅读体验。

同时，你也可以在 GitHub 上关注此项目的发展，如果您对此项目有什么意见，大可在 GitHub 上 Fork 它并且修改，然后向我们提交 Pull Request。
